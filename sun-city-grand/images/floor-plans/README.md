# The Grand Floor Plan Images

Authoritative master list source: Billy's typed collection (cross-referenced with leolinda.com).

The Grand was built in two phases. Some popular plans were redesigned in Phase 2 with new numbers, and a few names (Borgata, Valencia, Hacienda) appear in more than one collection at different sizes. Always match by **model number**, not just name.

## Filename convention

`{MODEL}.jpg` — model number only, no leading zeros, no spaces.

Examples: `9210.jpg` (Iris), `P9105.jpg` would be wrong (use `9105.jpg`), `E9251.jpg` would be wrong (use `9251.jpg`).
